## Getting Started

Prerequisites for installing and running the application locally.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/andrewgerez/clean-archcat.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Docker setup
   ```
   docker-compose up -d
   ```
4. Run the local server
   ```
   npm run dev
   ```
